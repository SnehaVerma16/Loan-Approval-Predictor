# 🏦 Loan Approval Prediction System

A Machine Learning web application that predicts whether a loan application is likely to be **Approved** or **Rejected** based on applicant details.

This project is built using **Python**, **Scikit-learn**, **Gradio**, and **Random Forest Classifier**, and is deployed on **Render**.

---

## 🚀 Live Demo

🔗 **Live Application:**[ https://YOUR-RENDER-LINK.onrender.com](https://loan-approval-predictor-y2nu.onrender.com)

---

## 📌 Features

- Predicts Loan Approval Status
- Interactive Gradio Interface
- User-friendly Sliders and Radio Buttons
- Input Validation
- Reset All Functionality
- Responsive Web UI
- Deployed on Render

---

## 🧠 Machine Learning Model

- Algorithm: Random Forest Classifier
- Dataset: Loan Approval Dataset
- Accuracy: **98.01%**
- Model saved using Joblib

---

## 📊 Input Features

- Number of Dependents
- Education
- Self Employed
- Annual Income
- Loan Amount
- Loan Term
- CIBIL Score
- Residential Assets Value
- Commercial Assets Value
- Luxury Assets Value
- Bank Asset Value

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/SnehaVerma16/Loan-Approval-Predictor.git
```

Go to the project folder

```bash
cd Loan-Approval-Predictor
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
